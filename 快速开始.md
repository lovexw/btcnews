## 🎯 三步部署

### 第一步：安装依赖
```bash
npm install
```

### 第二步：登录 Cloudflare
```bash
npm run login
```
浏览器会打开授权页面，点击"Allow"即可。

### 第三步：部署
```bash
npm run deploy
```

完成！你会看到类似输出：
```
✨ Success!
Published btc-news-aggregator-2025
  https://btc-news-aggregator-2025.xxxxx.workers.dev
```

## 🎊 首次使用

### 1. 打开部署的网址
复制上面返回的 URL，在浏览器中打开。

### 2. 触发首次抓取
访问：`你的URL/api/refresh`

等待几秒钟，页面会显示抓取结果。

### 3. 查看新闻
回到首页，刷新页面，就能看到抓取到的新闻了！

## 🆘 遇到问题？

### 页面空白
访问 `/api/refresh` 手动触发抓取

### 部署失败
```bash
wrangler logout
wrangler login
npm run deploy
```

### 想修改关键词
编辑 `src/index.js` 第 35-45 行，然后重新部署

### 想修改更新频率
编辑 `wrangler.toml` 第 17 行的 cron 配置，然后重新部署
